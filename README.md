# Pet Adoption Management System
The Pet Adoption Management System is a C# console application designed to assist animal shelters in managing pets up for adoption. This program showcases advanced concepts of object-oriented programming such as inheritance, abstraction, interfaces, polymorphism, composition, constructors, and access modifiers. Through a terminal-based menu system, users have the ability to add, view, update, search, and remove pet records.
